# 0.1.5 — 2026-05-23

## Changed

- **`Dockerfile`**: build stage restored to `oven/bun:1.3` (reverts the `node:24-slim` workaround from 0.1.4); `bun run build` replaces `npm run build`; OCI description label updated to the canonical tagline.
- **`package.json`**: all scripts migrated from `tsx <script>` to `bun run <script>`; `devcheck.config.json` adds `tsx` to the depcheck ignore list since the shebang lines use it but bun invokes scripts natively; `files[]` trimmed to remove stale entries (`changelog/`, `AGENTS.md`); `start:stdio` and `start:http` updated to `bun ./dist/index.js`.
- **`server.json`**: `runtimeHint` corrected from `"node"` to `"bun"` on both stdio and http packages; `MCP_PUBLIC_URL` env var added to the http package block.
- **`manifest.json`**: description updated to the canonical tagline; `repository`, `homepage`, and `license` fields added.
- **`NOMINATIM_USER_AGENT`**: default updated to `nominatim-mcp-server/0.1.5` across `server.json` (both packages) and `README.md`.
