# 0.2.0 — 2026-05-24

Renames the project across every surface — repo, npm package, Docker image, tool names, and env vars — to bring everything under a single `openstreetmap` namespace.

## Changed

- **Repository** renamed from `cyanheads/nominatim-mcp-server` to `cyanheads/openstreetmap-mcp-server`.
- **npm package** renamed from `@cyanheads/nominatim-mcp-server` to `@cyanheads/openstreetmap-mcp-server` (old package deprecated on npm).
- **Tool prefix unified** — all 6 tools renamed from `nominatim_*`/`overpass_*` to `openstreetmap_*`:
  - `nominatim_geocode` → `openstreetmap_geocode`
  - `nominatim_reverse` → `openstreetmap_reverse`
  - `nominatim_lookup` → `openstreetmap_lookup`
  - `overpass_query_nearby` → `openstreetmap_query_nearby`
  - `overpass_query_bbox` → `openstreetmap_query_bbox`
  - `overpass_query_raw` → `openstreetmap_query_raw`
- **Env vars renamed** — `NOMINATIM_BASE_URL` → `OSM_NOMINATIM_BASE_URL`, `NOMINATIM_USER_AGENT` → `OSM_USER_AGENT`, `OVERPASS_BASE_URL` → `OSM_OVERPASS_BASE_URL`.
- **Docker image** renamed from `ghcr.io/cyanheads/nominatim-mcp-server` to `ghcr.io/cyanheads/openstreetmap-mcp-server`.
- **Hosted endpoint** moving to `https://openstreetmap.caseyjhand.com/mcp` (from `nominatim.caseyjhand.com`).
- **Source files** — all `nominatim-*.tool.ts` / `overpass-*.tool.ts` renamed to `openstreetmap-*.tool.ts`; shared helpers and test files follow the same convention.

## Migration

Update tool calls (`nominatim_*`/`overpass_*` → `openstreetmap_*`), env var names (`OSM_*`), Docker image reference, and `npx`/`bunx` package name. No behavioral changes.
